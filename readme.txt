NHR Plugin Update Manager | Hide Frequent Updates

== Description ==

Getting too many updates from plugins? Well, say no to unnecessary frequent updates. Get updates whenever you want (e.x. once monthly) and Enjoy!

`<?php echo 'Less Update, Enjoyable Life!'; ?>`


== Installation ==

Upload the NHR Plugin Update Manager plugin to your blog, activate it.

1, 2, 3: You're done!


== Frequently Asked Questions ==

= Does it remove WordPress admin notices? =

No. It hides plugin update notices for a certain time period. E.x. If you want to get updates in the first week of each month 
then it wont display plugin update notices on the other weeks.

= Does it require any other plugin =

No. It works as standalone plugin.
